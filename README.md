# 🧘‍♀️ Eventplattform för Yoga & Meditation

## Deployad miljö

https://avdrxxa.github.io/webbshop-2026-fe/

## 📌 Om projektet

Detta projekt är en responsiv och fullt funktionell eventplattform som riktar sig till personer med intresse för yoga, meditation och välmående. Plattformen gör det enkelt för användare att upptäcka, boka och delta i olika event inom dessa områden.

## 👥 Team

Projektet utvecklades i samarbete mellan:

* 3 frontend-utvecklare (inklusive mig)
* 3 backend-utvecklare

Vi arbetade agilt och samarbetade nära mellan frontend- och backend-teamen för att säkerställa en smidig och effektiv utvecklingsprocess.

## 🚀 Funktioner

* 🔍 Bläddra och sök bland event
* 📅 Boka och hantera event
* 👤 Användarkonton och autentisering
* 📱 Responsiv design för mobil, surfplatta och desktop
* 🧘‍♂️ Fokus på wellness (yoga, meditation, mindfulness)
* 👤 Admin kan skapa, updatera, ta bort event, men även hantera kundregister

## 🛠️ Teknologier

### Frontend

* HTML, CSS, JavaScript, Trello, Figma

### Backend

* Vercel, Postman, Github, Git, Trello
